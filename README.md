<h2 align="center">Controls</h2>
    
| KEY      | ACTION |
| ----------- | ----------- |
| 1,2,3,4      | Change Race Track       |
| X   | Increase AI training speed        |
  | Z      | Decrease AI training speed          |
| L   | Toggle Input lines        |
| R      | Reset car position and change car model     |
  | W      | Car Acceleration     |
| A   | Turn Car Left       |
  | D      | Turn Car Right       |
